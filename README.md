## What Is WebArc.One?

WebArc.One is my personal portfolio and lab — not a static 
resume page. It's a living record of everything I'm building, 
learning, and shipping as an indie developer.

It has four sections:

| Section | What's Inside |
|---|---|
| [Projects](https://webarc.one/projects) | Live products I've built and shipped |
| [Case Studies](https://webarc.one/case-studies) | Deep technical breakdowns of each build |
| [Journal](https://webarc.one/journal) | Real-time build log — updated as I ship |
| [Connect](https://webarc.one/connect) | Collaborations, DMs, and reach-outs |
